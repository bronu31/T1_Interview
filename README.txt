Приложение состоит из одной страницы: "/api/calculate/"
После последнего символа "/" необходимо ввести строку для который требуется вывести частоту символов, после чего будет предоставлен результат в формате JSON, в нём будут представлены все символы которые есть в строке, при условии что строка корректна

В строке запроса нельзя использовать символы / или %, данные символы используются для указания URL что ведёт к ошибке 400.

Некоторые спец символы такие как ; ? не возвращают ошибку, но прекращают считывание строки, из-за чего расччёты могут производится некорректно.

Приложение можно запустить из среды разработки или же использовать maven.

Jar файл можно запустить из консоли windows или linux напечатав комманду java -jar T1interview-0.0.1-SNAPSHOT.jar
Для запуска приложения требуется версия java не ниже 17
